# Ollama API

После запуска Ollama, модели доступны через API, по умолчанию на `http://localhost:11434`. Выполняя различные запросы на этот адрес, можно взаимодействовать с моделями, которые были загружены в Ollama. Ниже приведены примеры использования API для различных задач.

- `GET /api/tags` - Получить список всех доступных моделей
