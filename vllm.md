# vllm

`vLLM` - это высокопроизводительная библиотека для обучения и инференса больших языковых моделей (LLM), разработанная компанией NVIDIA. Она оптимизирована для работы на GPU и поддерживает распределенное обучение, что позволяет эффективно использовать ресурсы при работе с моделями, состоящими из миллиардов параметров. `vLLM` обеспечивает высокую производительность и масштабируемость, что делает его популярным выбором для исследователей и разработчиков, работающих с LLM.
