# Конвертация моделей LLM

Конвертация моделей LLM - это процесс преобразования модели из одного формата в другой, что позволяет использовать модель в различных средах и с различными инструментами. Например, модель, обученная в формате PyTorch, может быть конвертирована в формат GGUF для использования с `llama.cpp`. Конвертация может включать в себя изменение структуры модели, квантификацию весов и активаций, а также оптимизацию модели для конкретного оборудования.
